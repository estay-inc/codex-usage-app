# Contributing

Thank you for contributing.

1. Fork the repository and create a focused branch.
2. Keep changes small and avoid adding analytics or credential access.
3. Run `./scripts/test.sh` before opening a pull request.
4. Explain the behavior change and how it was tested.

Please do not include ChatGPT tokens, Codex logs, account identifiers, email
addresses, or other personal data in issues, screenshots, fixtures, or pull
requests.

By contributing, you agree that your contribution is licensed under the MIT
License used by this repository.
